# 📱 Operaciones App

Aplicación móvil desarrollada en Android Studio que permite realizar operaciones matemáticas y conversiones de unidades mediante una interfaz sencilla e intuitiva.

## 🚀 Características

- Inicio de sesión con validación de usuario.
- Operaciones matemáticas básicas:
  - Suma
  - Resta
  - Multiplicación
  - División
- Conversión de unidades.
- Interfaz moderna utilizando Material Design.
- Navegación mediante pestañas (TabLayout y ViewPager2).

## 🛠️ Tecnologías Utilizadas

- Java
- Android Studio
- XML
- Material Design Components
- ViewPager2
- Fragment

4. Ejecutar la aplicación en un emulador o dispositivo Android.
